Task Management System with MERN Stack

This TODO List application helps users manage tasks efficiently. Built using the MERN stack, it allows users to add, edit, delete, and mark tasks as completed.

It features a modern React UI, a secure backend with Node.js and Express, and MongoDB for database storage.

With a responsive design, users can access their tasks from any device.

Organize your tasks effortlessly. A simple yet powerful task manager built with modern web technologies.
