AI-Powered Breast Cancer Prediction System

This project is a machine learning-based Breast Cancer Prediction System developed in PyCharm using Tkinter GUI. It utilizes multiple classification algorithms to analyze medical data and predict the presence of breast cancer.

The system supports various machine learning models such as Random Forest, Support Vector Machine (SVM), K-Nearest Neighbors (KNN), and Decision Trees, allowing for high-accuracy predictions.

The interactive Tkinter GUI provides a user-friendly interface where users can input relevant medical parameters, process data, and receive real-time predictions.

Early detection saves lives. AI and machine learning are transforming healthcare by providing accurate and efficient diagnoses.

The model was trained on the Wisconsin Breast Cancer Dataset (WBCD) and optimized for high precision and recall scores, minimizing false-negative predictions.
